Google Glass application to fly an AR.Drone quadcopter.

Prerequisites

  • In your computer, install Android Studio or Android Developer Tools (ADT).
  • In Google Glass, install (deploy) Launchy, so you can easily launch other installed Android applications from the Glass Settings.
